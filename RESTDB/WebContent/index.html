Total Inventory

Two new packages have been added (inventory and util). Inventory will allow you to manage computer parts in the PC_PARTS table. In the util package, a ToJSON class has been added to convert database resultset into JSON array.

One API resources was created in part 4:
http://localhost:8080/RESTDB/api/v1/inventory - This url will return all the rows PC_PARTS table.

Inventory by brand

Two of the API resources are for searching on the brand but but done differently:
The first one is by @QueryParam:
http://localhost:8080/RESTDB/api/v2/inventory?brand=nvidia
The second one is by @PathParam:
http://localhost:8080/RESTDB/api/v2/inventory/nvidia

The third api resource search by brand and item number using @PathParam:
http://localhost:8080/RESTDB/api/v2/inventory/nvidia/970

Posting to database

Introducing the POST method, we will be using POST to insert into the PC PARTS database table. There is going to be two examples. One using data binding (object mapping) The other way (2) is the use JSONArrays and JSONObjects. Both ways are valid, choose which one you want to go with.
